Description

A list item component used to display a summary of a single DICOM study. It provides key metadata at a glance and serves as an interactive element for selecting a study from a list.

Design System Alignment

This component utilizes the design system's color palette for states (default, hover, selected), typography for hierarchical information display, and spacing scale for internal padding and layout.

Accessibility Features

Keyboard Support

Component is focusable via Tab key. Can be activated with Enter/Space (when used as a button or link).

Screen Reader Support

Semantic HTML (e.g., using `` tags within a `

    `) allows screen readers to announce the component as an interactive list item. All text content is readable.

Focus Management

A clear, high-contrast focus ring appears on keyboard focus, using the `:focus-visible` pseudo-class.

Aria Attributes

For selected items, `aria-current="true"` or `aria-selected="true"` should be applied via JavaScript to inform screen readers of the state.

Color Contrast

Text colors meet WCAG AA contrast ratios against their backgrounds in all states (default, hover, selected).
