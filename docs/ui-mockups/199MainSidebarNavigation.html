Component Variations

Variant Name : expanded

The default, full-width state of the sidebar, showing both an icon and a text label for each navigation item. This provides maximum clarity and is suitable for wider application windows.

Variant Name : collapsed

A compact, icon-only state that saves horizontal screen space. The text labels are hidden, and the sidebar shrinks to its minimum width. Tooltips should be used to show the label on hover for usability.
