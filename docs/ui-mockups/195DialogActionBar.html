This area represents the content of a dialog. The action bar below provides the final actions for this dialog.

Component Variations

Confirm & Cancel

Standard layout for confirmation dialogs, featuring a primary (affirmative) and a secondary (dismissive) action.

OK Only

Used for informational dialogs where only a single acknowledgment action is required.

State: Disabled Action

Demonstrates the action bar when the primary action is disabled, for example, due to invalid form input in the dialog.
