Description

A comprehensive panel for displaying, filtering, and potentially editing hierarchical DICOM metadata. It includes a search/filter bar and displays tags in collapsible groups.

Design System Alignment

This organism utilizes the design system's color palette, typography, spacing, and foundational components like form inputs and buttons to create a cohesive data display unit.

Responsive Specifications

Mobile Adaptations

  • N/A. Desktop application component.

Tablet Adaptations

  • N/A. Desktop application component.

Desktop Enhancements

  • Clear hover and focus states for all interactive elements.
  • Scrolling content area for large metadata sets.

Accessibility Features

Keyboard Support

Tab to navigate between search bar, group headers, and metadata items/inputs. Enter/Space to toggle groups.

Screen Reader Support

Group headers are buttons with `aria-expanded` attributes. Editable fields are standard inputs.

Focus Management

Visible focus indicators on search bar, group headers, and editable inputs.

Aria Attributes

  • aria-expanded for group headers
  • aria-label for search input

Color Contrast

All text and UI elements meet WCAG AA contrast requirements.
