FormGroup Component

This component combines a label, an input control, and an error message into a single, cohesive unit.

Component Variations

Default State (TextInput)

Standard input field with a label. This is the default appearance for data entry.

Error State (TextInput)

Indicates a validation error. The label, border, and error message adopt the error color.
Password must be at least 12 characters.

Default State (ComboBox)

A standard dropdown selection field, styled to match other form controls.

Error State (ComboBox)

Indicates that a selection is required but has not been made.
This field is required.
