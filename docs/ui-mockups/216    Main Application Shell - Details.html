Accessibility Features

Keyboard Navigation

Users can navigate sidebar items using Tab and Arrow keys. The main content area is the next item in the tab order. The collapse button is keyboard accessible.

Screen Reader Support

The sidebar is wrapped in a `

` landmark. The main content area is in a `
` landmark. Active navigation items use `aria-current='page'`. Icons have accessible text labels.

Focus Management

Focus is clearly indicated with a visible focus ring on all interactive elements. When a new view is loaded in the content area, focus should be programmatically moved to its main heading.

Aria Attributes

  • aria-label for icon-only buttons
  • aria-current='page' for active navigation link
  • role='navigation'
  • role='main'

Color Contrast

All text and icons meet WCAG 2.1 AA contrast ratios against their backgrounds.
