Description

A contextual toolbar that provides users with all necessary tools for DICOM image manipulation, navigation, and interaction. It includes grouped actions and responsively moves less critical tools into an overflow menu on smaller viewports.

Design System Alignment

The toolbar uses the design system's icon-only button components, color palette for states (default, hover, active), spacing system for layout, and adaptive strategies for responsiveness.

Responsive Specifications

Mobile Adaptations

  • N/A (Desktop-first application).

Tablet Adaptations

  • On viewports narrower than 840px, less critical tools (e.g., Annotate, Measure, Print) are hidden from the main toolbar.
  • A 'More' button (overflow menu) becomes visible, containing the hidden tools in a dropdown list.

Desktop Enhancements

  • All tools are visible and directly accessible for quick interaction.
  • Clear hover and focus states provide rich feedback for mouse and keyboard users.

Accessibility Features

Keyboard Support

All toolbar buttons and the preset selector are focusable and operable using Tab, Shift+Tab, Enter, and Space keys. Arrow keys are used for the preset selector's options.

Screen Reader Support

The toolbar is wrapped in a `

` element with `aria-label`. All icon-only buttons have a descriptive `aria-label`. The preset selector has a visually hidden label.

Focus Management

A clear, high-contrast focus ring (`:focus-visible`) is applied to all interactive elements, ensuring keyboard users can always identify their location.
