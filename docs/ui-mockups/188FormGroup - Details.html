Description

A FormGroup encapsulates a label, a form control (like TextInput or ComboBox), and an associated error message. It manages the layout, spacing, and state changes (e.g., error state) for a single form field, ensuring accessibility and consistent styling.

Design System Alignment

This component uses the design system's typography for labels and errors, color tokens for states (primary for focus, error for validation), and spacing tokens for internal layout. It is the standard composite for building forms.

Screen Reader Support

Focus Management

A clear, high-contrast focus ring is applied to the control on `:focus-visible`.

Aria Attributes

  • aria-invalid="true" is set on the control in the error state.
  • aria-describedby is used to link the control to its error message.

Color Contrast

Label and error text colors meet WCAG AA contrast ratios against the background.
