Accessibility Features

Keyboard Navigation

Logical tab order from the password input to the 'Unlock' button, then to the 'Log Out' button. The 'Esc' key should not dismiss the dialog. Focus is trapped within the dialog.

Screen Reader Support

The dialog has `role='dialog'` and `aria-modal='true'`. The title is read out upon appearing. The password field has an associated label. Error messages are linked to the input via `aria-describedby`.

Focus Management

Initial focus is set on the password input field. Focus is contained within the dialog until it is dismissed by a successful unlock or logout.

Aria Attributes

  • role='dialog'
  • aria-modal='true'
  • aria-labelledby='lock-screen-title'
  • aria-describedby='password-error-message'

Color Contrast

All text and interactive elements meet WCAG 2.1 AA contrast ratios.
