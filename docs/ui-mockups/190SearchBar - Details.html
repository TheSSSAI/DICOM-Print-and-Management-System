Description

A composite input component used for searching and filtering lists of data. It combines a text input with icons for search and clearing, and an optional button to trigger advanced filter options.

Design System Alignment

Utilizes the design system's tokens for form inputs (border, focus state), colors (primary, neutrals), typography (body), and spacing. It assembles atomic components like TextInput, Button, and Icon into a functional unit.
