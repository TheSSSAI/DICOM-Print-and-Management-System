DICOM Viewer

An organism component for displaying and interacting with DICOM images.

Default (Single Viewport)

The standard interactive view. A toolbar provides tools, a thumbnail strip on the left allows series navigation, and the main viewport displays the selected image with metadata overlays.

Variant: Side-by-Side

The `.dicom-viewer--side-by-side` modifier class changes the viewport container to a two-column grid, allowing for the comparison of two different series or images.

State: Loading

The `.dicom-viewer--loading` class activates a skeleton loader overlay in the viewport. This provides immediate feedback that content is being fetched, improving the user experience.

State: Annotating

The `.dicom-viewer--annotating` class changes the viewport cursor to a crosshair. The active tool in the toolbar is highlighted with the `.is-active` class.
