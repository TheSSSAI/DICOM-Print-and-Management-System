Description

A standardized container for action buttons typically placed at the bottom of a dialog or modal. It provides consistent layout, spacing, and button alignment for actions like 'Confirm', 'Cancel', or 'OK'.

Design System Alignment

This component utilizes the 'Button' atomic component and the system's spacing tokens to ensure visual consistency with the rest of the application.
