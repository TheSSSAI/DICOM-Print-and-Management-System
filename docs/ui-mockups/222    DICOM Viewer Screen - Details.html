Responsive Specifications

Desktop Layout

  • Three-column fluid grid: Thumbnails (fixed-width), Viewer (flexible), Metadata (resizable).
  • Toolbar is anchored to the top.
  • Optimized for standard desktop and widescreen monitors.

Tablet Adaptations

  • N/A. This is a desktop-only application.

Mobile Adaptations

  • N/A. This is a desktop-only application.

Accessibility Features

Keyboard Navigation

Logical tab order from Toolbar -> ThumbnailStrip -> DICOMViewer -> MetadataPanel. Tools on the toolbar should have shortcuts (e.g., Ctrl+Z for Zoom).

Screen Reader Support

All panels and interactive elements have accessible names. The DICOMViewer component itself will require a custom Automation Peer to describe the image content (e.g., 'Axial CT Image 5 of 120').

Focus Management

A clear, high-contrast focus ring is applied to all interactive elements (toolbar buttons, thumbnails, metadata fields).

Aria Attributes

  • aria-label for icon buttons.
  • aria-busy='true' for the viewer during the loading state.
  • role='region' for distinct panels.

Color Contrast

All text in the MetadataPanel and tooltips meets WCAG 2.1 AA contrast ratios. The viewer itself is exempt as it displays clinical image data.
